# Metrics Converter

A simple web-based tool for converting between length and weight units.

## How to Use
- Enter your value.
- Select the units to convert from and to.
- Click "Convert"!

## Features
- Length: cm, m, km, ft, in
- Weight: kg, g, lb, oz
